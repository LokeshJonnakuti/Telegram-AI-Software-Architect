To set up the Telegram Translator Bot, follow these steps:
1. Create a new Telegram bot using BotFather on Telegram and obtain your bot token.
2. Replace 'YOUR_TELEGRAM_BOT_TOKEN_HERE' in src/telegram_bot.py with your actual bot token.
3. Install the required dependencies by running: pip install -r requirements.txt
4. Run the bot by executing: python -m src.telegram_bot
